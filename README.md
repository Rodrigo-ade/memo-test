# memo-test
 Juego

## Instalacion
Ejecutar `npm install` 

###### Testing
Ejecutar `npm run servidor-abrir` para abrir el servidor.
Ejecutar `npm run cypress-abrir` para abrir cypress.
Ejecutar ``npm run cypress-consola-abrir` para abrir cypress en la terminal.
**Para que cypress funcione es necesario abrir el servidor primero.*
